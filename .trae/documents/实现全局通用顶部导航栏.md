# 全局通用顶部导航栏实现方案

## 组件设计
1. **组件结构**：
   - 主容器使用弹性布局实现水平导航
   - 每个导航项为独立的胶囊按钮
   - 支持默认导航项和自定义扩展

2. **功能特性**：
   - 默认包含四个导航项：简介、电影、游戏、动漫
   - 支持通过 `navItems` prop 自定义导航项
   - 每个导航项包含 `label` 和 `href` 属性

3. **交互效果**：
   - 椭圆形长胶囊样式
   - 鼠标移入长胶囊时，轮廓亮起效果
   - 鼠标移入导航文字时，添加变换效果

## 实现步骤
1. **更新 `index.tsx`**：
   - 定义组件接口和默认导航项
   - 实现组件渲染逻辑
   - 添加自定义扩展支持

2. **更新 `style.module.css`**：
   - 添加导航容器样式
   - 实现椭圆形长胶囊样式
   - 添加鼠标移入效果

3. **样式设计**：
   - 使用 Tailwind CSS 实现基础样式
   - 添加 CSS 过渡效果增强用户体验
   - 确保响应式设计

## 预期效果
- 水平排列的椭圆形导航胶囊
- 鼠标移入时胶囊有轮廓亮起效果
- 导航文字在鼠标移入时有动画效果
- 支持自定义扩展导航项

## 代码示例
```tsx
// index.tsx
interface NavItem {
  label: string;
  href: string;
}

interface TopNavProps {
  navItems?: NavItem[];
}

const TopNav = ({ navItems }: TopNavProps) => {
  const defaultNavItems: NavItem[] = [
    { label: '简介', href: '/about' },
    { label: '电影', href: '/movies' },
    { label: '游戏', href: '/games' },
    { label: '动漫', href: '/anime' }
  ];

  const items = navItems || defaultNavItems;

  return (
    <nav className={styles.navContainer}>
      {items.map((item, index) => (
        <a
          key={index}
          href={item.href}
          className={styles.navItem}
        >
          {item.label}
        </a>
      ))}
    </nav>
  );
};
```
